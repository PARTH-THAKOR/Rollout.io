# Contributing to Rollout.io

We love your help! Contributing is easy:

1.  **Found a bug?** Open an issue here on GitHub.
2.  **Have a feature idea?** Open an issue to discuss it.
3.  **Want to fix something?** 
    *   Fork the repo.
    *   Make your changes.
    *   Send a Pull Request (PR).

That's it! We'll review your PR as soon as possible.
